**🐍 Galaxy Snake**
Galaxy Snake is a modern take on the classic Snake game built using HTML, CSS, and vanilla JavaScript. The game features a responsive grid-based board, smooth controls, score and time tracking, and persistent high scores.

**🚀 Features**
- Classic Snake gameplay with keyboard controls
- Responsive grid-based game board
- Score, high score (stored using localStorage), and timer
- Start and Game Over modals
- Clean and maintainable UI using CSS root variables
- Full-screen animated video background with dark overlay

**🛠 Tech Stack**
- HTML5 – Structure
- CSS3 – Styling with custom properties (:root variables)
- JavaScript – Game logic and state management

**🎮 Controls**
1. Arrow Keys – Move the snake
2. Eat food to increase score
3. Hitting the wall ends the game

**🧠 JavaScript Logic**
The JavaScript manages dynamic grid creation, snake movement, food generation, collision detection, keyboard input handling, score updates, and game state control. The code is structured for clarity and easy future expansion.

- Clone the repository
- Open index.html in your browser
- Click Start Game and play
